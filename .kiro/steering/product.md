# Product Overview

Sulbeef Representações is a static website built with Jekyll. The site serves as a web presence for a Brazilian representation company.

## Key Characteristics

- Static site generator approach for simplicity and performance
- Brazilian Portuguese language (pt-BR)
- Hosted on AWS S3 with CloudFront CDN
- Timezone: America/Sao_Paulo
- Currently marked as noindex (not indexed by search engines)
